This is a vanilla JS project, wich contains a list of products in a js file. 
Once you select any or some of them, you can access the cart page with details about the products you just selected, there, is possible to change the quantity, o just delete a specific product or all of them.
All the changes affect to product total, total amount and total units.
Aditionally the changes are persisted in local storage.
No installations required, just open the index.html file in your browser.
